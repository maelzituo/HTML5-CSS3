Principais Formatações

Negrito / Destaque

Nessa frase temos Um termo em Negrito usando a tag B (não semântica).

já nessa frase, temos um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

nessa frase, temos um termo em Itálico usando a tag i (não semântica).

já nessa frase, temos um Termo em Itálico usando a tag EM (semântica).

Texto Marcado

podemos criar também um texto marcado, usando a tag MARK.

e no outro parágrafo, também temos um termo marcado no final.

Texto Grande e Pequeno

estamos criando um texto com um termo grande, e um termo pequeno usando a tag BIG e SMALL.

Texto Deletado

podemos criar um texto como excluido para indicar que ele deve ser lido mas não considerado, usando a tag DEL.

Texto Inserido ou Sublinhado

també podemos criar um texto como inserido para indicar que ele foi adicionado depois, usando a tag INS.

existe também o sublinhado com a tag U (não semântica).
